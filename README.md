## Description

Groseries app as example of a simple [Nest](https://github.com/nestjs/nest) TypeScript implementation.

## Installation

```bash
$ npm install
```

Then run migrations with:

```bash
$ npm run migrations-run
```

> Migrations on test are run automatically by [TypeORM](https://github.com/typeorm/typeorm)

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testing

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
